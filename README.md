Estructura del programa:
gestor_paises.py <-- Main
paises.csv <-- Datos en archivo csv
utilidades.py 
busquedas.py
filtros.py
ordenamiento.py
estadisticas.py
validaciones.py
gestion.py
(El resto de archivos son funciones que ayudan al desarrollo correcto del programa)

Al dar inicio o ejecutar el archivo gestor_paises.py se iniciara un archivo csv, en caso de no existir este se creara
con los encabezados. A continuación se le mostrara al usuario un menu de 12 opciones en un bucle donde para terminar la ejecu-
ción del programa se debe elegir salir. Es un buen momento para aclarar que el programa esta hecho para evitar que un mal in-
greso de datos "rompa el programa", por lo tanto ni en el menu ni al agregar datos cualesquiera al csv el programa sufrira un
error fatal.

El archivo esta hecho para gestionar países, este es muy intuitivo y puedes elegir opciones como la 10 o la 11 que corresponden
a agregar o editar países correspondientemente y probar como funciona el programa, en cualquier caso se te mostrara una serie de instruccion a seguir según lo que quieras hacer! En cualquier momento puedes entrar al archivo paises.csv y comprobar los cambios realizados. 

Sea como sea aca hay un video explicativo que te ayudará en caso de que necesites más instrucciones.
